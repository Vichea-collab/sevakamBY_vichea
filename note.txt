Best setup: use admin-managed ads, not provider posts directly.

Recommended model:

Provider posts: regular marketplace content
shown in search, provider detail, recommendations, feed-style sections
Admin ads/promotions: curated banner carousel
shown in the home promo slider
admin chooses which provider/service/campaign to feature
Best approach is a hybrid:

Admin creates the ad.
Admin can optionally link that ad to:
a provider
a service
a category
an internal target page
Home page reads only active ads from admin data.
Clicking the ad opens:
provider detail
filtered search page
provider post page
or a special promo landing
What admin ad should contain:

title
subtitle/description
imageUrl
badgeLabel like Limited offer
ctaLabel
targetType like provider, service, category, search
targetValue
startAt
endAt
isActive
sortOrder
optional city/category targeting
Good future upgrade:

allow only Elite providers to request promotions
but admin still approves/publishes them
that keeps subscription value connected to promotions without losing quality control
So my recommendation is:

do not make the carousel auto-read provider posts
make it an admin-managed promotions system
optionally let admin link promotions to provider content
